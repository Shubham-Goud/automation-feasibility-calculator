# Automation Feasibility Calculator

This project is a decision-support tool that helps teams evaluate whether a business process is worth automating.

## Features
- Calculates monthly manual cost
- Calculates monthly automation cost
- Shows cost difference and ROI
- Provides automation recommendation
- Exports results to Excel

## Technologies Used
- Python
- Streamlit
- SQLite
- Pandas

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
